O dataframe da biblioteca pandas possui o método plot que gera gráficos de dispersão (scatter plot) com o parâmetro kind='scatter':


import numpy as np
import pandas as pd

np.random.seed(1)
data = np.random.randn(500, 2)
df = pd.DataFrame(data, columns=list('XY'))

df.plot(
    x='X',
    y='Y',
    kind='scatter',
    s=50,
    alpha=.5,
    figsize=(6,5)
)
