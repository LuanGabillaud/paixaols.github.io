Gráfico de linha

O código abaixo gera um simples gráfico de linha a partir de dados contidos em duas listas.


import matplotlib.pyplot as plt

x = [1,2,3,4,5]
y = [2,5,3,5,7]

plt.plot(x, y)

É possível customizar o estilo de linha com o parâmetro linestyle e os símbolos com o parâmetro marker.


import matplotlib.pyplot as plt

x = [1,2,3,4,5]
y = [2,5,3,5,7]

plt.plot(x, y, linestyle='--', marker='o')

O método plt.grid habilita as linhas de grade que podem auxiliar na Visualização do gráfico.


import matplotlib.pyplot as plt

x = [1,2,3,4,5]
y = [2,5,3,5,7]

plt.plot(x, y, linestyle='--', marker='o')
plt.xlabel('Eixo x')
plt.ylabel('Eixo y')
plt.grid(linestyle='--')

Customização dos eixos e legendas é exemplificada no código abaixo.


import matplotlib.pyplot as plt

x = [1,2,3,4,5]
y1 = [2,5,3,5,7]
y2 = [7,4,5,3,2]

plt.plot(x, y1, linestyle='--', marker='o', label='Linha 1')
plt.plot(x, y2, linestyle='-.', marker='s', label='Linha 2')
plt.xlabel('Eixo x')
plt.ylabel('Eixo y')
plt.grid(linestyle='--')
plt.legend()
