Estilos do matplotlib

A biblioteca matplotlib possui vários estilos de formatação para customização rápida dos gráficos. Uma lista completa dos estilos existe na documentação da biblioteca. Aqui, exemplos com o estilo ggplot:


import matplotlib.pyplot as plt
plt.style.use('ggplot')

x = [1,2,3,4,5]
y1 = [2,5,3,5,7]
y2 = [7,4,5,3,2]

plt.plot(x, y1, '-o', label='Linha 1')
plt.plot(x, y2, '-s', label='Linha 2')
plt.legend()
plt.title('ggplot')
