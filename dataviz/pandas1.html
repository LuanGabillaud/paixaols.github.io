Pandas

O dataframe da biblioteca pandas possui o método plot que possibilita criar gráficos com facilidade. Para a criação do dataframe vamos usar o código


import numpy as np
import pandas as pd

x = np.arange(0, 10, 0.2)
y = np.random.randint(low=0, high=11, size=50)
df = pd.DataFrame({'col x':x, 'col y':y})

As primeiras linhas do meu dataframe são:

col x col y
0 0.0 6
1 0.2 2
2 0.4 10
3 0.6 0
4 0.8 8

É possível plotar cada coluna em função do índice do dataframe da seguinte forma


df.plot()

Com os parâmetros do método plot é possível determinar quais colunas do dataframe serão plotadas e ainda customizar o gráfico:


df.plot(
    x='col x', y='col y',
    xlabel='Eixo x',
    ylabel='Eixo y',
    title='Gráfico de linha',
)
